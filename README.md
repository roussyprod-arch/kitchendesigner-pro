# 🏠 KitchenDesigner Pro

Application web complète de conception de cuisines avec outils 3D, e-commerce et gestion de projets.

## 🎯 Caractéristiques

- ✨ Concepteur 3D intégré
- 🛍️ Boutique e-commerce complète
- 👥 Système d'authentification secure
- 📁 Gestion des projets
- 💳 Paiement Stripe
- 📧 Système de notifications email
- 🔍 Moteur de recherche avancé
- ⭐ Système d'avis et favoris
- 📱 Design responsive
- 🚀 Déploiement Docker automatisé

## 📋 Prérequis

- Node.js >= 16.0.0
- npm >= 8.0.0
- MongoDB 5.0+
- Docker & Docker Compose (optionnel)
